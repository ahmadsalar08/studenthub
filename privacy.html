Kya data collect hota hai?

Jab tum StudentHub use karte ho, hum kuch basic information collect karte hain:

  • Name & Email — account banane ya contact form pe
  • Reading history — kaunse articles padhe (improve karne ke liye)
  • Device & Browser info — technical issues solve karne ke liye
  • Location (approximate) — sirf country level, exact nahi

💡 Hum kabhi sensitive data nahi maangenge — passwords hints, financial info, ya government IDs.

⚙️

Yeh data kaise use hota hai?

Tumhara data sirf website ko better banane ke liye use hota hai:

  • Tumhari preferences ke hisaab se articles suggest karna
  • Website ki performance improve karna
  • Important updates ya replies ka email dena
  • Spam se community protect karna

Hum tumhara data advertisers ya kisi bhi baahri party ko sell nahi karte.

🍪

Cookies kya hoti hain?

Cookies chhoti files hoti hain jo browser mein save hoti hain. Hum sirf zaroori cookies use karte hain:

  • Session cookies — logged in rakhne ke liye page reload pe
  • Preference cookies — dark mode ya settings yaad rakhne ke liye

Marketing ya tracking cookies bilkul use nahi karte. Browser settings se kabhi bhi clear kar sakte ho.

🤝

Data kisko share hota hai?

Tumhara personal data kisi ko share nahi hota. Sirf yeh exceptions hain:

  • Supabase — sirf encrypted database storage ke liye
  • Vercel — website hosting ke liye zaroori
  • Legal requirement — agar court officially request kare

🔒 In services ko sirf minimum zaroori data diya jata hai, woh bhi encrypted form mein.

✅

Tumhare rights kya hain?

Tum apne data ke baare mein yeh actions le sakte ho:

  • Dekho — hum ne kya data save kiya hai
  • Edit karo — galat info correct karwao
  • Delete karo — account aur sab data permanently hatao
  • Object karo — kisi specific data use se mana karo

Koi bhi request ke liye contact form use karo. 7 din mein jawab milega.

🛡️

Data kitna secure hai?

  • Saara data HTTPS (SSL) se encrypt transfer hota hai
  • Database Supabase Row Level Security se protected
  • Passwords hashed hote hain — plain text mein kabhi nahi
  • Regular security checks kiye jate hain

Security breach hone pe hum turant affected users ko email karenge.

Koi sawal hai? 💬

Privacy ke baare mein koi bhi concern ho toh seedha contact karo.
